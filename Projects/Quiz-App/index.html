Quiz Application

Customizable trivia app built with Express and Open Trivia Database

JavaScript Express Web Development API Integration
Personal Project

Overview

Built a full-featured quiz application allowing users to select categories, difficulty, and question types, take timed quizzes, view results, and create custom quizzes for others. The project integrates Express for API functionality and Open Trivia Database for default questions.

Functionality

  • Customizable Quizzes: Users can select categories, question counts, difficulty, and type of questions.
  • Timed Quizzes: A timer tracks how long it takes to complete a quiz.
  • Results Summary: Users see their score, time, and a full answer key upon completion.
  • Quiz Builder: Users can create and publish custom quizzes for others to attempt.

Key Takeaways

This project strengthened my skills in full-stack JavaScript development, API integration, and user experience design. It demonstrates the ability to create interactive web applications with dynamic content and user-generated features.
