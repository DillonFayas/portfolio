Unit Pricing Database

A streamlined database solution for construction material pricing estimation

JavaScript HTML CSS SharePoint Git Database Design
Software Contract

Situation

A construction company needed a centralized way to store and access unit pricing data. Their existing process was slow and inconsistent across departments, making it difficult for estimators to prepare accurate bids efficiently.

Task

I was responsible for designing and implementing a database solution and building an accessible interface that non-technical staff could use for estimating material costs. The solution needed to work within the company's existing SharePoint infrastructure.

Actions

I created a structured unit-pricing database and built a user-friendly SharePoint frontend. I configured lists, forms, and workflows to streamline data entry and retrieval, ensuring that the system would be intuitive for staff members who weren't technically inclined.

I used HTML, CSS, and JavaScript to write the application, building custom interfaces on top of SharePoint's framework. Throughout the project, I used productivity tools like Git for version control and Trello to track work and manage tasks.

Since I was new to the SharePoint platform at the start of this project, I learned the platform's tools quickly to meet project expectations. I adjusted the design based on user feedback, iterating on the interface to ensure it met the team's needs.

Key Features

  • Centralized database for all unit pricing information
  • Advanced search and filtering capabilities
  • Historical pricing data tracking and analysis
  • Supplier comparison tools
  • Data export functionality for estimation documents
  • Role-based access control for data management

Results

The new system standardized pricing data and reduced estimation time. Team members could access accurate pricing without exhaustive searching, improving consistency across bids. The centralized approach eliminated the departmental inconsistencies that had previously slowed down the estimation process.

This project strengthened my ability to learn unfamiliar frameworks quickly and adapt to new development environments. Working within SharePoint's constraints taught me how to build effective solutions using the tools available, rather than requiring ideal conditions.
